API: URL Shortener microservice

This microservice shortens the url provided by the user to the web browser addess. Below is how to use this service:
    1-) Enter the web link to the explorer address (see below for example entries)

    Example usage:

    https://https://url-shortener-microservices.herokuapp.com/newuser/www.google.com

    Example output:

    { "shortened_url": www.google.com, "original_url": 'l2kxja' }
    2-) If entry is invalid - the service will provide an error.
