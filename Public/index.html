API: URL Shortener microservice

This microservice shortens the url provided by the user to the web browser addess. Below is how to use this service:
    1-) For new short url creation to database, enter the web link to the explorer address.

      Example:

      https://url-shortener-microservices.herokuapp.com/new/www.bbc.com

      Example output:

      {"original_url":"https://www.bbc.com","short_url":"https://url-shortener-microservices.herokuapp.com/14lu55"}
    2-) For redirecting from a short url to the original website, enter the newly generated short link to the explorer address.

      Example:

      https://url-shortener-microservices.herokuapp.com/14lu55

      This would redirect to https://www.bbc.com
